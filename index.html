개발자 진단

Supabase 실제 연결 진단

버튼 동작만으로는 서버 연결을 확정할 수 없어요. Auth와 DB에 실제 데이터가 생성되는지 확인합니다.

아직 전체 서버 연결 진단을 실행하지 않았어요.

계정 상태

Google 또는 Kakao 계정을 연결해도 기존 아기 기록은 그대로 유지됩니다.

OAuth 계정은 접근 권한 계층일 뿐이며, records의 실제 owner는 family workspace입니다.

현재 익명 사용자로 사용 중입니다. 카카오 계정을 연결하면 다른 기기에서도 가족 기록을 이어서 사용할 수 있습니다.

Kakao/Google 로그인이 작동하려면 Supabase Dashboard에서 Provider와 Redirect URL 설정이 필요합니다.

가족/아기 서버 구조

가족/아기 구조는 Phase 4 로그인 기능으로 넘어가기 위한 준비 단계예요. 현재는 익명 사용자 기준으로 안전하게 연결해 둡니다.

가족/아기 구조를 확인하면 현재 서버 연결 상태를 볼 수 있어요.

가족 연결

가족 코드를 사용하면 새 기기에서 같은 family workspace에 연결하고 서버 기록을 이 기기에 복원할 수 있어요. 연결 전 현재 로컬 데이터는 자동 백업됩니다.

연결 전 현재 로컬 기록과 설정을 자동 백업합니다.

가족/아기 서버 구조를 확인하면 가족 코드와 연결된 기기 목록을 볼 수 있어요.

서버에 저장된 기록을 불러와 이 기기의 기록과 비교할 수 있어요. 병합 전에는 현재 기기 데이터가 자동 백업됩니다.

서버 기록을 불러오면 병합 미리보기를 만들 수 있어요.

현재는 익명 사용자로 서버 저장을 준비하고 있어요. 나중에 로그인 기능이 추가되면 여러 기기에서 더 안전하게 사용할 수 있어요.
